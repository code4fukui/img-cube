# img-cube

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A 3D image cube component built with Three.js.

## Demo
https://code4fukui.github.io/img-cube/

## Features
- Rotating 3D cube display
- Web component implementation
- Customizable size and appearance

## Usage
To use the `img-cube` component, include the `img-cube.js` file and add the `<img-cube>` element to your HTML:

```html
<script type="module" src="./img-cube.js"></script>
<img-cube></img-cube>
```
